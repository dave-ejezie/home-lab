# Phase 2: M365 & Google Workspace Support

This directory contains labs and documentation for Phase 2 of the Enterprise IT Support Homelab, focusing on Cloud administration competencies.

## Labs Included

1. **[01-M365-Admin-Centre](./01-M365-Admin-Centre)**: M365 Admin Centre Fundamentals (Licence management, user creation).
2. **[02-Hybrid-AD-Connect](./02-Hybrid-AD-Connect)**: Hybrid Identity Sync (Entra Connect).
3. **[03-MFA-Reset](./03-MFA-Reset)**: MFA Reset (A critical Service Desk skill).
4. **[04-Shared-Mailboxes](./04-Shared-Mailboxes)**: Shared Mailboxes & Distribution Lists.
5. **[05-Conditional-Access](./05-Conditional-Access)**: Conditional Access Troubleshooting.
6. **[06-Google-Workspace](./06-Google-Workspace)**: Google Workspace Admin Basics.

## Learning Objectives

- Manage user licenses in M365 (assign, remove, identify unused).
- Execute secure MFA resets for locked-out users.
- Differentiate between Teams and distribution groups.
- Create and manage shared mailboxes.
- Perform basic administration tasks within Google Workspace.
- Configure Hybrid Identity by syncing on-premises AD to Entra ID.
